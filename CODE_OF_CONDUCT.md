# Code of Conduct

We want Ambient Codex to be a useful, welcoming project.

Be respectful, discuss the work rather than the person, accept constructive
feedback, and avoid harassment, discrimination, threats, or publishing another
person's private information. Maintainers may edit or remove contributions and
restrict participation when behavior harms the project or its community.

Report conduct concerns privately through the repository's security advisory
channel. Include only the information needed to investigate; do not publish
sensitive personal details in an issue.
